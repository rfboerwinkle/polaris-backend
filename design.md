# Frontend

- Web-based only, primarily targeting desktop resolutions (1280x720 or 1920x1080)

# Accounts

- Have username and password
- History of played games available to be viewed
- Checkbox for opting into data collection

# Single Player Games

- Can be played without accounts
- Can choose 1-5 bots to play against
- Can set bot difficulty with 3 options (easy, medium, hard)
- Can be played offline
- Not ranked

# Multiplayer Party Games

- Require account to create or join
- One person creates game
- 1-5 others can join game
- Anyone in game can assign themselves to any color that isn’t already taken
- Anyone in game can add or remove bots
- Not ranked

# Multiplayer Assigned Games

- Require accounts to join
- Randomly assign with someone else in the queue
- Ranked with a modified ELO
